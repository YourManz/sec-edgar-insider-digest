## How to Use This Digest

1. **This is not a buy list.** It's a watch list — insider buying is one signal among many.
2. **Check the Form 4 yourself.** Go to [SEC EDGAR](https://www.sec.gov/cgi-bin/browse-edgar) and search the ticker. Verify the filing type is "P" (open-market purchase), not "A" (grant) or "M" (option exercise).
3. **Layer your own thesis.** Insider buying is most powerful when it confirms something you already believe about the business.
4. **Track the outcome.** Note whether the cluster held up. Over time you'll develop a feel for which patterns correlate with recoveries.

---

## Methodology

- Source: SEC EDGAR Form 4 filings, pulled via public EDGAR full-text search API
- Filter: Open-market purchases only (type = "P") — excludes grants, RSU vests, option exercises
- Cluster rule: 3+ distinct insiders (different names) buying within the same rolling 7-day window
- No screening for size: a $5,000 purchase by a director counts the same as a $500,000 purchase by the CEO in the cluster filter, but we annotate the value so you can weight accordingly
